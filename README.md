README.md

Nhóm thực hiện project AI Multi-Agent Search dựa trên Project 2 của CS188 Berkeley.  
Mục tiêu của nhóm là nghiên cứu và cài đặt các thuật toán tìm kiếm nhiều tác nhân để điều khiển Pacman trong môi trường có ghost đối kháng và ngẫu nhiên.

Trong project này, nhóm tập trung vào:

- Xây dựng Reflex Agent
- Cài đặt Minimax, Alpha-Beta Pruning và Expectimax
- Thiết kế hàm đánh giá trạng thái cho Pacman
- Phân tích hiệu quả và sự khác nhau giữa các thuật toán
